The Zamen WordPress Plugin ensures your WordPress blog is running optimally on the Zamen News App.

== Description ==
Zamen WordPress plugin lets you sync your WordPress blog with Zamen Platform.

By using the Zamen WordPress Plugin, you receive:

* Your posts to be publish on Zamen platform and Zamen App without the need to open RSS.

* Comments on Zamen platform are synced with your wordpress native comments.

* Add the push notification message that user will get when you publish a new post.


THINGS YOU NEED TO KNOW:

* The main purpose of this plugin is to ensure that your posts are synced with Zamen platform without the need to open RSS also to sync the comments between Zamen platform and your wordpress native comments system.
